❓ دليل المساعدة

ستجد هنا شرحًا مبسطًا وواضحًا لكل صفحة من صفحات النظام، مع توضيح الوظائف الأساسية لكل جزء. إذا واجهت أي مشكلة أو لديك استفسار، راجع الشرح أدناه أو تواصل مع الدعم.

🏠 لوحة التحكم

  • تعرض ملخص سريع لأهم الإحصائيات (عدد الطلاب، الإيرادات، أحدث الفواتير، ...).
  • رؤية سريعة لحالة السنتر.

💵 الكاشير

  • إنشاء فواتير جديدة للطلاب.
  • إضافة اشتراكات أو ملازم فردية.
  • البحث عن طالب بسرعة.

🧾 الطلاب المشتركين

  • استعراض جميع الفواتير.
  • البحث والتصفية حسب الاسم أو الفرقة أو النوع.
  • فتح تفاصيل الفاتورة وطباعتها.

🧍‍♂️ فواتير فردية

  • إدارة ملازم أو محاضرات فردية للطلاب غير المشتركين.
  • إمكانية البحث والتصفية.

💳 الاشتراكات والأسعار

  • ضبط أسعار الكورسات والملازم.
  • إضافة أو تعديل اشتراكات جديدة.
  • تحديد الأسعار لكل فرقة.

👤 إدارة المستخدمين

  • إضافة وتعديل وحذف المستخدمين.
  • التحكم في الصلاحيات.
  • تفعيل أو تعطيل الحسابات.

📊 التقارير

  • استخراج تقارير الأرباح.
  • توزيع الطلاب.
  • تحليل الإيرادات والمصروفات.

💸 المصروفات

  • تسجيل المصروفات اليومية.
  • استعراض ملخص المصروفات.
  • البحث والتصفية.

⚙️ الإعدادات

  • ضبط بيانات السنتر (الاسم، العنوان، الشعار).
  • إعدادات النظام العامة.

❓ دليل المساعدة

  • شرح كل صفحة ووظائفها.
  • نصائح للاستخدام الأمثل.
